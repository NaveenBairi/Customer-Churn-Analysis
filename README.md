This Power BI project focuses on analyzing telecom customer data to understand customer churn behavior.
It explores customer demographics, account details, service usage, and billing information.
The project aims to identify key factors that influence why customers leave the service.
Data cleaning and preprocessing were performed to ensure accurate analysis.
DAX measures were created to calculate churn rates and customer segments.
Interactive dashboards visualize churn trends across tenure, charges, and customer types.
The analysis highlights high-risk customer segments prone to churn.
Insights help in understanding the relationship between pricing and customer retention.
The dashboard supports data-driven decision-making for retention strategies.
Overall, the project demonstrates the effective use of Power BI for business analysis.
